UVXY Options Research

Systematic studies on UVXY put strategies across VIX futures curve regimes (contango/backwardation).

This repository contains backtests and diagnostics on long and short UVXY put strategies, with all results adjusted for historical reverse splits. Performance is evaluated using FWD 90% PnL (near-expiration outcomes).

Reports

  • Long Put UVXY — FWD 90% PnL Analysis
    Transformed view of the same base dataset (sign-flipped from short-put PnL).
  • Short Put UVXY — FWD 90% PnL Analysis
    Cross-sectional summaries by delta, DTE, and term-structure slope buckets.
  • Short Put UVXY — Edge Diagnostic & Regime Breakdown
    Robustness checks and regime-dependent behavior.
