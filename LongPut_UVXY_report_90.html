Data provenance and study design: these results are built from ThetaData option-chain records together with UVXY market quotes. The base dataset is generated by creating one short-put candidate per trading day across multiple delta targets and expiration tenors from January 1, 2019 through December 2025. This LongPut report is a transformed view of that same base dataset.

Method note: in this LongPut report, PnL_fwd_pts_90_mediana is defined as -1 x the original ShortPut PnL_fwd_pts_90_mediana. The "90" point means evaluation at approximately 0.9 x DTE (near expiration). This is an endpoint-at-horizon analysis, not a full path analysis of the trade. So intra-trade mark-to-market behavior, interim drawdowns, and time-under-water are not reflected here. Figures are filter-level cross-sectional summaries over rows with non-null PnL, and they do not model sizing, overlap, slippage/fees, or margin usage.

Corporate actions note: UVXY reverse splits in the sample include 2021-05-26 (1-for-10), 2023-06-23 (1-for-10), 2024-04-11 (1-for-5), and 2025-11-20 (1-for-5). Dataset values are adjusted for these reverse splits. A validation in ONE (OptionNetExplorer) was performed on selected trades outside split-event windows, and the results match the dataset outputs.

Glossary: M2/M1-1 = (M2 / M1) - 1, where M1 is the front-month VIX future and M2 is the second-month VIX future. Positive values indicate contango (M2 > M1), negative values indicate backwardation (M2 < M1). UVXY is a short-term VIX futures ETP with daily rolling exposure from M1 into M2, so this slope is a key driver: contango usually implies negative roll yield (structural decay in UVXY), while backwardation usually implies positive roll yield and can support sharp UVXY rises during stress regimes.
